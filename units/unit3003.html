Your First Web Lab

Congratulations, you’ve been commissioned by Banksy, a street artist, to create a small portfolio webpage that showcases 6 of his art works.

Banksy would like the webpage to look like this:

The Life of a Programmer

You may be thinking to yourself, “Hmmm, I haven’t learned how to put images on a webpage, I haven’t learned how to center text on a webpage, and I haven’t learned how to put a link on a webpage.”

You’re right! It’s important to know that this is typical and normal in the life of a programmer. We are often asked to do things that we haven’t learned just yet.

What’s important in this lab is that you begin to learn how to look up programming resources, adapting their information to the problem at hand!

Start with what you know

Of all the things we’re being asked to do, we do know how to structure a basic webpage with text, so let’s start there to get some momentum started.

1. Create a new text file called banksy.html

2. Give this file the basic page structure elements of a webpage.

3. Add text to the webpage that says “Banksy, a street artist” and “Born in Bristol, England circa 1974.” Do your best to format the text with the right size and font.

Is all your text on the same line?

You haven’t learned how to put text on different lines yet, so there’s a good chance it’s all running together on one line right now. Is this happening on your webpage?

4. Head to Google.com and try searching for an answer to your problem. In programming, putting text on a new line is often referred to as a “line break” or “carriage return.” But you wouldn’t know these terms yet! So let’s search using terms you might guess at:

    How do I put text on a new line in html?
    I need my html text on different lines
    Basic HTML text formatting

Choose a search or two that seems good to you. Then, take a few minutes to read through the top 3 or 4 results returned.

5. Experiment with some of the potential solutions you’re finding. Try them in your webpage and see if you can get the two texts on different lines.

Checkpoint 1

Click the answer tabs to see how your code stacks up.  

There are many potential solutions, but these are probably the most commonly expected!

<!doctype html>
<html>
    <head></head>
    <body>
        <!-- this solution uses paragraph tags of <p> to create line breaks -->
        <p><font face="Arial" size="20">Banksy, a street artist</font></p>
        <p><font face="Arial">Born in Bristol, England circa 1974.</font></p>
    </body>
</html>

<!doctype html>
<html>
    <head></head>
    <body>
        <!-- this solution uses a line break tag of <br> to create a line break -->
        <font face="Arial" size="20">Banksy, a street artist</font><br>
        <font face="Arial">Born in Bristol, England circa 1974.</font>
    </body>
</html>

If you didn’t find one of these answers, that’s okay! What’s important is that you start cultivating a mindset that is comfortable not knowing all of the answers upfront. As you search for more answers over time, you’ll develop a strength and confidence in the ability to find answers to the new problems that you’ll face on a daily basis.

Let's proceed using the code from Answer 1a.

Is your text centered?

Banksy’s request seems to have the text centered in the page. If you’re working with the code from Answer 1a above, then our text is not centered.

6. Use Google to search for “how do I center my html paragraph text?”

7. Find and use an attribute for the paragraph tag that we can use to center the text.

Checkpoint 2

Click the answer tabs to see how your code stacks up.  

There are many potential solutions, but these are probably the most commonly expected!

<!doctype html>
<html>
<head></head>
<body>
    <!-- this uses a paragraph attribute of <p align="center"> to center text -->
    <p align="center"><font face="Arial" size="20">Banksy, a street artist</font></p>
    <p align="center"><font face="Arial">Born in Bristol, England circa 1974.</font></p>
</body>
</html>

8. Use Google to search for how you can add images to an html webpage.

9. Try adding the banksy_1.gif to your webpage.

If you’re successful in getting the image to appear on the screen, you should notice that it’s gigantic compared to its size in the Banksy request.

10. Use Google to search for an image attribute you can use to resize the image. Try different sizes and see if you can find one that will get you close to the relative size in the Banksy request.

11. Add the other images to your webpage, resize them, and do your best to put them on the same lines as the Banksy request.
