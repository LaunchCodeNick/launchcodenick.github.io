Quiz Answers

1. What is the difference between <%= %> and <% %> tags? Use examples.

Click on the Answer tab to see the answer!

// You should use <%= %> when directly printing something to the page like this:
<%= "this is text I want to print" %>

// You should use <% %> when you want to run more complex code like this:
<%
int i = 3;
int j = i * 2;
out.println(j);
%>

2. What will print out if you run the following code? Explain why this is.

out.println(7/2.0);

The command will print out 3.5

This is because when you run a calculation between an integer and a double, the resulting answer will be a double.  Doubles can always hold an integer, but not vice versa.

int age = 41;
int agedividedby2 = age / 2; // would store 20

// if 21 == 20 ...
if ((age - agedividedby2) == agedividedby2) { 
    out.println("even");
} else {
    out.println("odd");
}

The Mod Command (%)

There is another command that makes the odd/even algorithm easier. That command is called ‘mod’, which is used with a % sign.

Mod will give you the remainder from any division command. So if you divided 10 by 3, the remainder would be 1. 10 % 3 will yield 1 in Java.

To solve the even or odd problem, even numbers will have 0 remainder when divided by 2. Odd numbers will have a remainder of 1.

int age = 41;

if ((age % 2) == 0) {
    out.println("even");
} else {
    out.println("odd"); 
}
