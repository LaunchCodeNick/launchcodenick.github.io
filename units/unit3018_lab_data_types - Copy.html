Numeric Data: Integers

Let’s start with one of Java’s most common numeric data types: the integer.

Integers are whole numbers (no decimals) that can be positive or negative.

For example, -1000,-475, 0, 1, 2, 200, 2000.

We can declare an integer using the keyword ‘int’ and assign a value to it using one equal sign.

Edit the Java portion of your web page to the following:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    int myvariablename = 0;
    out.println (myvariablename);
%>

5. Refresh your page to see the output.

You should see a zero appear on your screen!

6. You can use many of the math operators you’d expect to work, and you can also use parenthesis to structure an order of operation.

Try the following code:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    int myvariablename1 = 0 + 1 + 2;
    int myvariablename2 = 1 + 2 * 3;
    int myvariablename3 = (1 + 2) * 3;
    int myvariablename4 = 4 / 2;
    out.println (myvariablename1); out.println("<br>");
    out.println (myvariablename2); out.println("<br>");
    out.println (myvariablename3); out.println("<br>");
    out.println (myvariablename4); out.println("<br>");
%>

7. Remember that an integers do not have decimals. What do you think will happen when we try something like 5 / 2?

Try the following code:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    int myvariablename1 = 5 / 2;
    out.println (myvariablename1); 
%>

Be careful when dividing integers if the exact value matters to you. If you do need the decimals, it may be wise to leverage another numeric data type such as a ‘double’.

Numeric Data Types: Doubles

A double is a data type that can hold a much larger number and ones with decimals.

8. Let’s use the following code and see some doubles in action:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    double myvariablename1 = 5 / 2; // this line will not yield 2.5
    double myvariablename2 = 5.0 / 2.0;
    double myvariablename3 = 3.0 * 3.0;
    out.println (myvariablename1); out.println("<br>");
    out.println (myvariablename2); out.println("<br>");
    out.println (myvariablename3); out.println("<br>");
%>

Notice that even though we are using doubles, the numbers themselves must be written with a decimal to get operators such as division correct!

Mixing Doubles and Integers

9. Let’s test what will happen if you mix doubles and integers in the same calculation:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    double myvariablename1 = 5.0 / 2;
    double myvariablename2 = 5 / 2.0;
    double myvariablename3 = 3.0 * 3;
    out.println (myvariablename1); out.println("<br>");
    out.println (myvariablename2); out.println("<br>");
    out.println (myvariablename3); out.println("<br>");
%>

When mixing numeric data types, the code will generally use and resolve to the type that can include the other. For example, an integer can’t represent a double of 0.5, but a double can represent any integer. So when mixing the two, a double will be the outcome.

Boolean Data Type

Sometimes, all we need to store in a variable is whether something is true or false. We call this a boolean.

10. Let’s see some booleans in action with the following code:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    boolean isSmart = true;
    boolean isRichLikeBillGates = false;    
    out.println (isSmart); out.println("<br>");
    out.println (isRichLikeBillGates); out.println("<br>");
%>

Strings

You’ve already been exposed to Strings in some of your Hello World examples. A String is some amount of text we need to store, such as “Hello World!” Strings are wrapped in double quotation marks.

11. Let’s see some strings in action with the + operator. When we add strings together, we call that concatenation.

<%
    out.println("<p>This is our Data Type Lab!</p>");            
    
    String firstWord = "hello";
    String secondWord = "world";
    String allWords = firstWord + secondWord;
    out.println (firstWord); out.println("<br>");
    out.println (secondWord); out.println("<br>");
    out.println (firstWord + secondWord); out.println("<br>");
    out.println (allWords); out.println("<br>");
%>

Don’t expect that Java will put a space between Strings that you add together!

Converting Numbers to Strings

12. It’s not often that you’ll want to convert a number into a String. Usually, this occurs when you want to print a number to the screen.

If we try something like out.println(“this is a zero: “ + 0), it will work but what Java is really doing is converting the 0 to a String of “0” and adding them together.

Use the following code to see an example: ~~~ html <% out.println(“<p>This is our Data Type Lab!</p>”);

out.println ("this is a zero: " + 0); 
out.println ("this is a zero: " + 0 + 0 + 0 + 0 + 0);  %> ~~~

13. You can force Java to do any sort of numeric calcuations before Java converts numbers into a string by using parentheses.

Try this:

<%
    out.println("<p>This is our Data Type Lab!</p>");            
        
    out.println ("this is a zero: " + 0); 
    out.println ("this is a zero: " + (0 + 0 + 0 + 0 + 0)); 
%>

Converting Strings to Numbers

What if you have to go the other way and convert a String to a number? You may have to do that because user input might be captured as a String, but actually contains a number.

14. Try this code:

<%
    out.println("<p>This is our Data Type Lab!</p>");            

    String hamburgerQuantityText = "4";
    int hamburgerQuantity = hamburgerQuantityText;
    hamburgerQuantity = hamburgerQuantity * 2; // double my order!
    
    out.println ("I have this many hamburgers: " + hamburgerQuantity);     
%>

You should get an error that says, “cannot convert from String to int.”

We will need to use special commands in Java to convert numbers into Strings.

15. Try this code instead:

<%
    out.println("<p>This is our Data Type Lab!</p>");            

    String hamburgerQuantityText = "4";
    int hamburgerQuantity = Integer.parseInt(hamburgerQuantityText);
    hamburgerQuantity = hamburgerQuantity * 2; // double my order!    
    out.println ("I have this many hamburgers: " + hamburgerQuantity);    

    String hamburgerPriceText = "4.95";
    double hamburgerPrice = Double.parseDouble(hamburgerPriceText);
    out.println (", each one costs me: " + hamburgerPrice + 
                 ", unless I have a 10% coupon which would make them: " + 
                 (0.90 * hamburgerPrice));    
%>

Integer.parseInt() and Double.parseDouble() are commands you may need in a later lab!
