Common Questions

Why are the images offset? They don’t appear to be perfectly aligned.

They appear offset for two reasons:

1. The images themselves contain whitespace around the edges. You can edit your image tags with the following attribute to see:

    <img src="banksy_4.jpg" border="3">

2. Not all the images are exactly the same size unless you specified both the height and width of them all. When images of different sizes are put next to one another, the images will end up aligning to an edge, such as the bottom, or the top. In this case, some of the images align on the bottom.

Activity

Use Google to search for basic html tags. Find a guide for beginners and scan some of the tags listed to see what HTML can do.

You’ll find a wide variety of resources out there. Some are far too simple and will only have 5 tags for you. Others are too exhaustive and will try to list every tag that exists. What tends to be the most helpful at this stage, is something in the middle.
