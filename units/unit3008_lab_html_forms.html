Web Forms

So far, you’ve learned two types of input controls, a textbox and a submit button. In HTML, there are a few more types of input controls that you can use. Let’s try using some of them to see how they work!

1. Take a few minutes to use google.com to look for a list of HTML input controls.

You should find a few different types of controls that include textboxes, checkboxes, radio buttons, and regular buttons.

Samburgers needs your help!

Congratulations, a local business called Samburgers has asked you to help them implement a small online ordering form. Here’s what they would like their website to look like:

2. Start with what you know to build some momentum. Create a file called cart.html . You should be able to create the “Shopping Cart”, “Price, Quantity, Item”, and “Calculate Your Order” lines. Pay attention to font size and face.

3. Add the divider lines you see on the page. These are called “horizontal rules” and you might not have stumbled upon the tag for it yet. If not, use Google to find the correct HTML tag to draw the line.

4. Add a submit button to the page.

Hamburger Line Item

Let’s work on adding the hamburger line.

5. You should be able to create the price and quantity textboxes and the word “Hamburger” for the line.

A good solution should have textboxes that aren’t too big or too small!

6. Make the first textbox readonly since it contains the price per hamburger. This isn’t something we want the customer to edit.

7. Add the checkboxes and text for Cheese and Special Sauce.

8. Make sure each input control has a name.

Without a name, the values of input controls won’t be passed to the querystring when the form is submitted.

Challenge Question

What is the difference between the checkboxes we used for the Hamburger line item and the radio buttons we used for the Fries line item?

Checkboxes are great when there is a list of options and the user can select any combination of choices including zero, one, or several.

Radio buttons are great when only 1 item in a group can be selected.  For example, payment type might be by cash, credit card, or check.  But it's usually not a combination of the three.

Set Vanilla as a default choice

16. See if you can find two ways to make Vanilla the default Ice Cream flavor on the shopping cart form.

If you didn't find two ways, you can click on the tabs here to discover them.

<!-- this answer just moves Vanilla to be the top item in the list 
     the top item in a list is selected as the default on a form -->

<select name="icecreamflavor">
    <option value="vanilla">Vanilla</option>                
    <option value="chocolate">Chocolate</option>
    <option value="strawberry">Strawberry</option>    
</select>

<!-- this answer keeps Vanilla at the bottom of the list 
     but adds an attribute and value of selected to make it the default -->

<select name="icecreamflavor">                
    <option value="chocolate">Chocolate</option>
    <option value="strawberry">Strawberry</option>    
    <option value="vanilla" selected="selected">Vanilla</option>    
</select>
