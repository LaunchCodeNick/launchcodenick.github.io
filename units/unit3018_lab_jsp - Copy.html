3. Rename cart.html to cart.jsp

Capture Querystring Parameters

In JSP, we can use the following convention to retrieve querystring parameters:

<% String x = request.getParameter("parameter_name"); %>

It’s important to know that in Java, these commands are case-sensitive! Upper and lowercase letters matter!

This will NOT work:

<% string x = REQUEST.GETParameter("parameter_name"); %>

Calculating a Total

Our final objective with this lab is to calculate the total $$ amount of the order.

In order to do that, we have to solve a few issues. The first problem is that to add up $ amounts, we’d love to have our variables stored as numeric data types. Unfortunately, our variables are currently stored as Strings because that’s what querystring values are.

9. Try changing one of your request.getParameter lines to the following, and then refresh your page:

int hamburgerQuantity = request.getParameter("hamburger_quantity");

You should get a compile error because Java won’t let you casually place a String text value into a numeric variable.

10. You can ask Java to convert a String to a numeric type with special commands. For a String to int conversion, use this:

int hamburgerQuantity = Integer.parseInt(request.getParameter("hamburger_quantity"));

Change all of your Quantity variables to int data types using Integer.parseInt.

11. Try changing your hamburger price variable to the float data type (because it can handle decimals) using Float.parseFloat.

float hamburgerPrice = Float.parseFloat(request.getParameter("hamburger_price"));

You should receive an error. Look at the error page and see if you can identify the “root cause”. Java is saying it found an error, or exception, because of an input string “$4.95”. As it turns out, we can’t convert “$4.95” to a numeric data type because it contains a dollar sign character! However, we can convert “4.95” to a numeric data type.

12. Remove all of the dollar signs from your HTML input controls. Try the code from line 11 again.

Yikes, you might get another error! Look at the error page again and see if you can identify the “root cause”. This time the error is different, and you should see a NullPointerException. Java can’t convert a null String into a numeric data type. When we first visit the web page, there are no querystring parameters because we haven’t submitted the form yet. We will need to add a little code to solve the problem.

13. We’re going to change our page so that we only try to store and calculate variables when the form has been submitted.

The question now is: how can we figure out when the form has been submitted? We’re going to add a name to our submit button and then check for it first with our Java code.

Change your submit button code to the following:

<input type="submit" name="submit" />

Now add an “if” statement that looks like the code below. An “if” statement will only run the code inside the { } braces if the statement is true. The != operator means “not equals.” So the statement below is saying only run our code if the querystring parameter for “submit” is not null (meaning not empty or missing).

if (request.getParameter("submit") != null) {
    float hamburgerPrice = Float.parseFloat(request.getParameter("hamburger_price"));
    int hamburgerQuantity = Integer.parseInt(request.getParameter("hamburger_quantity"));
    // put all the rest of your existing code here too, this is just a short code example!
}

14. Convert all of your price variables to float data types, and convert all of your quantity variables to int data types. Make sure to add null value checks!

15. Test your page. Try changing some of the values and submitting the form a few times.

Adding everything up

Now that our numeric variables are in numeric data types, and we are error free, we’re ready to calculate the total.

16. Make a float variable named ‘total’ and set it equal to 0.

17. Let’s add the following code to get our final total:

float total = 0;
total = total + (hamburgerPrice * hamburgerQuantity);
total = total + (friesPrice * friesQuantity);
total = total + (sodaPrice * sodaQuantity);
total = total + (icecreamPrice * icecreamQuantity);

if (hamburgerQuantity > 0) {
    if (hamburgerCheese.equals("on")) {
        total = total + 0.30f; // the f tells Java this is a float
    }
    if (hamburgerSauce.equals("on")) {
        total = total + 0.30f; // the f tells Java this is a float
    }
}

out.println("Order Total: $" + total + "<br>");                    
out.println("<br>");
