Code Growth

A small, but important point here: notice that it didn’t take a very big form or complex program to make a long page of code. In this course, we’re going to learn ways to reduce the amount of code we write to maintain efficiency, and we’ll also learn how to manage large blocks of code and functionality via separate pages and files.
