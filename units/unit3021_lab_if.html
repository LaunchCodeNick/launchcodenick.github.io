Working with Java Variables and If statements

In this lab, we’re going to do some work with variables and if statements to solve a numerical challenge.

Your program needs to declare an integer variable called ‘pennies’ and set it equal to 798.

Then, your program needs to calculate how to convert those pennies to a more efficient form of change in dollars and coins. For example, 798 pennies should receive 7 dollars, 3 quarters, 2 dimes, 0 nickles, and 3 pennies of change.

Although you should know this, here’s a reference table just in case:

Money Equals this number of pennies
Dollars 100
Quarters 25
Dimes 10
Nickles 5
Pennies 1

The final output of your program should not print a type of coin if the value is zero. For example:

798 pennies should be converted to the following change:

7 dollars
3 quarters
2 dimes
3 pennies

798 pennies should be converted to the following change:

7 dollars
3 quarters
2 dimes
0 nickles
3 pennies

Lab Setup

This lab will not contain multiple steps and checkpoints for you. Instead, please work with a partner to think through this problem!

You have learned all of the programming concepts necessary to complete this challenge. It may be a good strategy to solve this on paper before you write any code. How would you do this in real life without a computer?

One Hint

In case you are really struggling with this lab, at the 15-minute mark, you may take a look at this hint to see how to start your lab!

<!doctype html>
<head><title>Data Type Lab</title></head>
<body>
    <%
        int pennies = 798;  // the initial count of pennies              
        int penniesRemaining = pennies; // how many pennies still need to be accounted for 
        
        int dollars = penniesRemaining / 100;
        int penniesRemaining = penniesRemaining - (dollars * 100);        
    %>
</body>
</html>
