Quiz Answers

1. Write a simple HTML page that has the 4 essential markup tags for page structure.

Click on the Answer tab to see the answer!

<!doctype html>                         <!-- this is element 1 -->
<html>                                  <!-- this is element 2 -->
    <head>                              <!-- this is element 3 -->
        <title>Good Practice</title>    <!-- not required but a good practice -->
    </head>
    <body>                              <!-- this is element 4 -->
        Hello World!
    </body>
</html>

2. Explain what is wrong with the following code. How would you fix it?

Click on the Answer tab to see the answer!

There are 3 issues with this code.
1.  <font> needs an ending tag of </font> to come after Hello World!
2.  The font attribute of size should be inside of double quotation marks like so size="20".  This is not a hard error, but is a good practice.
3.  The closing <p> tag should be </p> instead.

Solution:
<p>
    <font face="Arial" size="20">Hello World!</font>
</p>

3. Imagine there is an HTML tag called Faketag. If Faketag has two attributes named Smile and Crash, and you wanted to set both to yes, what would your HTML tag look like?

Click on the Answer tab to see the answer!

<faketag smile="yes" crash="yes"></faketag>

4. What is different about these two blocks of code?

Click on the Answer tab to see the answer!

As far as HTML is rendered, nothing is different about these code examples.  The second one, however, is more readable to a human programmer because the indentation makes it easier to follow the sequence and nesting of tags.

5. In Examples 1 and 2 above, if you had to give someone instructions on how to indent their code to change it from Example 1 to Example 2, how would you explain it?

Click on the Answer tab to see the answer!

The essence of this answer should say something close to "Use a new indent or tab each time you nest another tag.  When you close a tag, align it under its opening tag."
