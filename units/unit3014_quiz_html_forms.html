Quiz Answers

1. Write the HTML code necessary to put a working textbox, checkbox, and dropdown list with two items on a page.

Click on the Answer tab to see the answer!

<!-- it's important that each control has a name -->
<!-- in order for them to work in any kind of form submission, they must be inside of a form tag -->

<form method="get">
    <input type="textbox" name="textbox_name" />
    <input type="checkbox" name="checkbox_name" />
    <select name="dropdown_name">
        <option value="option1">Option 1</option>
        <option value="option2">Option 2</option>
    </select>
</form>

2. In the following example, assuming this page is on a site named http://example.com/text.html what will the url look like when the form is submitted:

<input type="textbox" name="textbox1" value="ABCD" />
<form method="GET">
    <input type="textbox" name="textbox2" value="XYZ" />
    <input type="textbox" name="textbox3" value="123" />
    <input type="textbox" value="7890" />
    <input type="submit" />
</form>
 
Click on the Answer tab to see the answer!

textbox1 will not make it to the querystring because it's outside of a form.
The last textbox will not make it to the querystring because it has no name.
The beginning of a querystring starts with a ?
Each set of querystring name and values is separated with a &

Solution:
http://example.com/test.html?textbox2=XYZ&textbox3=123

3. How would you change the code example in question 2 to submit the form to Google.com?

Click on the Answer tab to see the answer!

<!-- to change where the form is submitted, you add an action attribute to the form tag -->

<form method="get" action="http://www.google.com/">
    <input type="textbox" name="textbox_name" />
    <input type="checkbox" name="checkbox_name" />
    <select name="dropdown_name">
        <option value="option1">Option 1</option>
        <option value="option2">Option 2</option>
    </select>
</form>

4. In your own words, what is the difference between a static and a dynamic web page?

Click on the Answer tab to see the answer!

There a probably a number of great answers to this, but hopefully you included the following:

A static web page looks the same each time you open it.  
HTML, CSS, and JavaScript are technologies used in static web pages.
Static web pages do not need web servers to run, although web servers are generally used to share static web pages across the Internet.

A dynamic web page can look different depending on multiple factors, including user input.  HTML, CSS, JavaScript and an additional back-end language such as Java are technologies used in dynamic web pages.
Dynamic web pages require a web server that can run the back-end code at page request time.
